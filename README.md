# vscode-codeskills

Public Codex skills repository.

## Available skill

- `shouxin-report`

## Install

Install from the repo path:

```bash
python3 /Users/daisy/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py \
  --repo Daisyhu668/vscode-codeskills \
  --path shouxin-report
```

Or install directly from the GitHub tree URL:

```bash
python3 /Users/daisy/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py \
  --url https://github.com/Daisyhu668/vscode-codeskills/tree/main/shouxin-report
```
